## Available functions

```answer2reviewers```: you can run `answer2reviewers(1)` to generate an answer for you first reviewer.

```whereIsXXX```: Run `whereIsRampal()`, `whereIsRobin()` or, `whereIsChristophe()` to find the location of your supervisors!

```noicePaletteGenerator```: Run `noicePaletteGenerator(n)` to get a palette of n colors!
